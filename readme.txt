ElementForge is a high-performance, modular library of premium widgets and templates built to supercharge the Elementor Page Builder experience.

== Description ==

ElementForge transforms your Elementor page building workflow with a meticulously crafted suite of advanced widgets and dynamic template fetching capabilities. From beautiful team profiles to advanced typography and SaaS-style pricing tables, ElementForge is engineered for performance.

= Key Features =
* **Component-Based Architecture:** Only the CSS and PHP you actively use on a page is loaded! We guarantee zero frontend bloat.
* **Modern React Dashboard:** Manage which widgets are active directly from a lightning-fast React backend panel.
* **Advanced Typography:** Build beautiful gradient headings.
* **SaaS Pricing Tables:** Construct engaging, interactive pricing layouts natively inside Elementor.
* **Instant Cloud Templates:** (Coming Soon) Seamlessly import beautiful readymade page templates directly into your Elementor canvas using our Smart Remote API!

== Installation ==

1. Upload the `element-forge` directory to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure you have the `Elementor` plugin installed and activated.
4. Navigate to the ElementForge tab in your WordPress dashboard to toggle your preferred widgets.
5. Open any page in "Edit with Elementor" to start building!

== Frequently Asked Questions ==

= Does this plugin slow down my website? =
No. ElementForge uses strict lazy-loading component injection. The CSS assets for a widget are strictly only enqueued if you drag that specific widget onto the page.

= Can I use it on free Elementor? =
Yes! ElementForge is built natively to extend Elementor Core without needing Elementor Pro.
